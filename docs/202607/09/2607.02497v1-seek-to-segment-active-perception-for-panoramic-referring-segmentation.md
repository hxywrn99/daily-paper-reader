## 摘要
现有的指代分割模型被动处理固定视角拍摄的静态图像，限制了其在具身人工智能中的应用，其中智能体必须在连续的360°环境中进行主动感知。为弥合这一差距，我们引入了一项新任务：主动全景指代分割（APRS）。在此设置中，智能体需要调整其观看方向（Δθ, Δφ）以探索360°环境，寻找用户指令指定的物体进行分割。为应对这一挑战性任务，我们提出了PanoSeeker，一种用于高效APRS的记忆增强智能体。与依赖启发式扫描不同，PanoSeeker将视觉语言模型（VLM）与EgoSphere（一种显式空间视觉记忆）相结合。通过逐步将连续的局部观测整合为统一的360°表示，EgoSphere使智能体能够规划高效且非冗余的搜索轨迹。一旦找到目标，智能体执行主动视角对齐并输出分割掩码。此外，我们整理了一个包含记忆时间线的专家标注搜索轨迹数据集用于监督微调，随后进行强化学习后训练以显式优化PanoSeeker的探索效率。在我们新建立的APRS基准上的大量实验表明，PanoSeeker实现了卓越的搜索效率和分割精度，显著优于改编的最先进基线方法。

## Abstract
Existing referring segmentation models passively process static images captured from fixed perspectives, limiting their applicability in Embodied AI, where agents must perform active perception in the continuous 360$^\circ$ environments. To bridge this gap, we introduce a novel task: Active Panoramic Referring Segmentation (APRS). In this setting, an agent is required to adjust its viewing direction ($Δθ, Δφ$) to explore the 360$^\circ$ environment, seeking the object specified by a user instruction for segmentation. To tackle this challenging task, we propose PanoSeeker, a memory-augmented agent for efficient APRS. Rather than relying on heuristic scanning, PanoSeeker integrates a Vision-Language Model (VLM) with EgoSphere, an explicit spatial visual memory. By progressively integrating sequential local observations into a unified 360$^\circ$ representation, EgoSphere enables the agent to plan efficient and non-redundant search trajectories. Once the target is found, the agent performs active viewpoint alignment and outputs the segmentation mask. Furthermore, we curate an expert-annotated search trajectory dataset with memory timelines for Supervised Fine-Tuning, followed by Reinforcement Learning post-training to explicitly optimize PanoSeeker's exploration efficiency. Extensive experiments on our newly established APRS benchmark demonstrate that PanoSeeker achieves superior search efficiency and segmentation accuracy, significantly outperforming adapted state-of-the-art baselines.

- **意义**：APRS将指代分割从被动静态设置扩展到主动动态探索，为具身AI中的视觉-语言导航与操作提供了更贴近现实的基准。

### 2. 方法论：核心思想、关键技术细节、算法流程（文字说明）

- **核心思想**：提出PanoSeeker——一种记忆增强的智能体，集成视觉语言模型（VLM）与显式空间视觉记忆EgoSphere。通过将连续局部观测逐步融合为统一的360°等距柱状图（ERP）表示，避免冗余搜索；结合监督微调（SFT）与强化学习（RL）优化探索效率；最终通过主动视角对齐与分割模块生成掩码。
- **关键技术细节**：
  - **EgoSphere**：显式空间视觉记忆。初始化一个零值的ERP画布M∈R^{H×W×3}，每步将当前视角V_t通过逆球心投影映射回ERP坐标并逐像素更新。同时加入三种视觉提示：当前FoV中心十字准星、30°间隔经纬网格、已探索轨迹，将搜索转化为360°画布上的“填空”任务。
  - **PanoSeeker动作预测**：基于VLM（Qwen3-VL-8B-Instruct）处理三个输入流：当前局部观测V_t、标注记忆图M_t、指令I。输出动作a_t∈{ (Δθ_t,Δφ_t), [STOP] }。
  - **SFT阶段**：使用专家标注的轨迹数据集，最小化负对数似然损失L = -E_{T~D_expert} Σ log p_θ(a_t | V_t, M_t, I)。
  - **RL阶段**：采用GRPO（Group Relative Policy Optimization），无需独立critic。定义奖励函数：每步效率奖励R_eff = γ[dist(V_{t-1},V*) - dist(V_t,V*)] - η，终端奖励R_term在成功时给予R_base+α·IoU，失败时惩罚-R_pen。GRPO目标最大化组内归一化优势下的策略梯度，并加入KL散度正则化。
  - **主动对齐与分割**：收到[STOP]后，VLM预测边界框B，然后利用SAM-3进行中心寻找、跟踪和分割，并执行最终角度调整使目标质心对齐光心，获得稳定分割掩码S。

### 6. 主要结论与发现

- PanoSeeker在APRS基准上全面优于所有对比方法：SR 75.4%，AS 4.8步，SPL 0.57，mIoU 55.8%。

- 记忆架构对比表明，几何一致的360°地图远优于纯文本或图像序列积累。

### 7. 优点

1. **任务创新**：首次提出APRS，将指代分割从被动静态转为主动探索，填补具身AI中主动视觉感知的空白。
2. **方法论设计巧妙**：EgoSphere利用逆球心投影将局部观测融合为统一全景画布，自然避免了冗余；视觉提示（十字准星、网格、轨迹）增强了空间推理。
3. **训练策略完善**：SFT+RL两阶段训练，SFT学习专家轨迹，RL显式优化效率，奖励函数考虑距离变化和IoU，逻辑合理。
4. **基准构建严谨**：数据来源多样（4,971场景），包含四种难度级别的指代表达，人工标注+SAM-3+VLM交叉校验，保证质量。评估指标多维。
5. **实验全面且消融充分**：对比多种范式（静态、启发式、主动VLM），消融组件和记忆架构，结果清晰。

### 8. 不足与局限

1. **环境简化**：基于全景图像模拟360°环境，而非真实3D场景（如点云或NeRF），忽略了深度、物理交互等；主动调整视角背后是离散化的30°步长，与实际连续运动存在差距。
2. **基准规模有限**：7,420样本（4,971场景），相比传统指代分割数据集（如RefCOCO等）仍然较小；场景分布可能偏向特定拍摄条件。
3. **缺乏真实机器人部署验证**：仅在模拟环境下评估，未在真实移动机器人或VR平台上测试泛化性（如光照变化、遮挡、动态物体）。
4. **未细分表达类型表现**：未报告不同复杂度（EGO vs MULTIHOP）下的独立性能，无法判断模型在困难子集上的鲁棒性。
5. **分割模块依赖SAM-3**：主动对齐阶段的SAM-3为冻结模型，可能受限于SAM的域外性能；且最终分割质量受定位精度影响。
6. **可重复性**：代码、模型权重、标注数据集未在论文中明确提供开源链接（仅有项目网站），可能影响实验复现。
