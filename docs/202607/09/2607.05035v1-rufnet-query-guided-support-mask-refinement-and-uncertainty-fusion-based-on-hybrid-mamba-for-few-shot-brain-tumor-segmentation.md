## 摘要
少样本脑肿瘤分割由于支持掩码噪声、支持与查询图像间的患者间变异性以及缺乏像素级置信度估计而仍具挑战性。本研究提出RUFNet，一种基于混合Mamba的少样本框架，结合了支持掩码细化和不确定性感知后验融合。为了以可管理的成本保持支持-查询依赖关系，RUFNet采用了线性复杂度的混合Mamba交互骨干。为了减少支持掩码噪声，注意力引导掩码细化模块（AGMR）使用查询特征重新校准支持掩码并提高原型一致性。为了处理模糊预测，不确定性感知后验融合模块（UAPF）估计像素级方差并自适应地平衡少样本预测与查询对齐先验。在脑肿瘤分割挑战（BraTS）2020数据集上，RUFNet在1-way 1-shot和1-way 5-shot设置下分别达到84.3%和86.1%的Dice系数，优于当前最先进方法。这些结果表明，混合Mamba交互、掩码细化和不确定性建模可以提高少样本医学图像分割的鲁棒性。官方实现代码可在https://github.com/hdy6438/RUFNet获取。

## Abstract
Few-shot brain tumor segmentation remains challenging due to noisy support masks, inter-patient variations between support and query images, and the lack of pixel-wise confidence estimation. This study proposes RUFNet, a Hybrid Mamba-based few-shot framework that combines support mask refinement with uncertainty-aware posterior fusion. To preserve support-query dependencies with manageable cost, RUFNet adopts a Hybrid Mamba interaction backbone with linear complexity. To reduce support-mask noise, an Attention-Guided Mask Refinement module (AGMR) uses query features to recalibrate support masks and improve prototype consistency. To handle ambiguous predictions, an Uncertainty-Aware Posterior Fusion module (UAPF) estimates pixel-wise variance and adaptively balances few-shot predictions with query-aligned priors. On the Brain Tumor Segmentation Challenge (BraTS) 2020 dataset, RUFNet achieves Dice coefficients of 84.3% and 86.1% in the 1-way 1-shot and 1-way 5-shot settings, respectively, outperforming the compared state-of-the-art methods. These results suggest that Hybrid Mamba interaction, mask refinement and uncertainty modelling can improve the robustness of few-shot medical image segmentation. The official implementation code is available at https://github.com/hdy6438/RUFNet.

### 核心思想

### 关键技术细节

#### （1）混合Mamba特征骨干（Hybrid Mamba Backbone）
- 基于HMNet（Hybrid Mamba Network），包含**支持重置模块（SRM）**和**查询隔离模块（QIM）**。
- SRM：周期性地将支持特征插入查询序列，防止支持特征遗忘，保持类先验。
- QIM：解耦查询内部像素间依赖，使每个查询像素主要从支持状态获取信息，减少查询内部变异干扰。

#### （2）注意力引导掩码细化模块（AGMR）
- 输入：支持特征 \(F_S\)、查询特征 \(F_Q\)、原始二值支持掩码 \(M_{\text{ori}}^S\)。
- 步骤：
  1. 通过交叉注意力计算支持到查询的语义匹配得分 \(A_{S \leftarrow Q} = \text{Softmax}(Q_S^T K_Q / \sqrt{d})\)。
  2. 用注意力重聚合查询证据得到查询增强支持特征 \(O_S = V_Q A_{S \leftarrow Q}^T\)，并与原始支持特征残差融合：\(\tilde{F}_S = F_S + \gamma W_O O_S\)。
  3. 将原始掩码嵌入连续特征空间 \(E_S = W_M M_{\text{ori}}^S\)。
  4. 拼接 \(\tilde{F}_S\)、\(F_S\)、\(E_S\)，经1×1卷积+BN+Sigmoid生成软化掩码 \(\hat{M}_S\)，保留边界和噪声区域的“部分前景”语义。

#### （3）不确定性感知后验融合模块（UAPF）
- 输入：查询域融合特征 \(F_u\)、查询对齐先验 \(M_{\text{prior}}\)（由细化后支持掩码投影得到）。
- 步骤：
  1. 通过特征提取器 \(\Phi\) 和预测头 \(\Psi\) 输出均值 \(\mu(x)\) 和对数方差 \(s(x)\)，方差 \(\sigma^2(x)=\exp(s(x))\)。
  2. 将方差映射为权重 \(w(x)=\exp(-\alpha\sigma^2(x))\)，单调递减。
  3. 后验融合：\(M_{\text{post}}(x)=w(x)M_{\text{meta}}(x)+(1-w(x))M_{\text{prior}}(x)\)，其中 \(M_{\text{meta}}(x)=\text{Sigmoid}(\mu(x))\)。
  4. 再经3×3卷积+BN+1×1卷积+Sigmoid得到最终输出 \(M_{\text{final}}\)，增强空间一致性。
