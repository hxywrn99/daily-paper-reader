## 摘要
现代视频目标分割（VOS）涉及跟踪和分割用户指定的目标。尽管近期方法在单目标场景中取得了显著性能，但将其扩展到多目标设置通常需要为每个单独目标复制单目标处理流程，导致帧率（FPS）下降，且延迟随目标数量增加而无界增长。基于Segment Anything 2（SAM2），我们提出SAM-MT，通过将模型转换为用于实时多目标视频分割的交互式框架来解决这一问题。SAM-MT使用显式查询来表示不同的个体目标，同时采用共享表示捕捉全局上下文。它利用解耦掩码注意力来保持个体身份独立，避免跨目标干扰，并使用稀疏记忆实现稳定的时序演化，同时结合处理遮挡和防止重叠的专门策略。SAM-MT成功地将延迟与目标数量解耦，实现了与单目标基线相当的实时速度（10个目标时>36 FPS），同时保持了SAM2鲁棒的视频分割性能。

## Abstract
Modern Video Object Segmentation (VOS) involves tracking and segmenting user-specified targets. While recent approaches have achieved remarkable performance in single-target scenarios, extending them to multi-target settings typically involves replicating the single-target processing for each individual object, resulting in reduced frame rates (FPS) with unbounded latency as target count increases. Built upon Segment Anything 2 (SAM2), we propose SAM-MT, which addresses this by transforming the model into an interactive framework for real-time Multi-Target video segmentation. SAM-MT uses explicit queries to represent different individual targets, in parallel with a shared representation for global context. It employs decoupled masked attention to keep individual identities distinct from cross-target interference, and sparse memory for stable temporal evolution, along with specialized strategies for occlusion handling and overlap prevention. SAM-MT successfully decouples latency from the number of targets, achieving real-time speed on par with single-target baselines (>36 FPS for 10 targets) while maintaining SAM2's robust video segmentation performance.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题**：现代视频目标分割（VOS）方法在单目标场景中表现优异，但扩展至多目标时，通常需要为每个目标独立复制单目标处理流程，导致计算成本和延迟随目标数量线性增长，帧率（FPS）急剧下降，无法满足实时交互需求。
- **背景**：基于空间-时间-记忆（STM）范式的SOTA方法（如SAM2、Cutie）依赖对象级记忆和传播，本质上是为单目标设计的。简单合并所有目标为统一掩码会丢失个体ID且形状不规则，模型难以跟踪。
- **动机**：实际应用（如自动驾驶）需要实时跟踪并分割多个目标，同时保持个体身份。因此，需要一种框架能在目标数增加时保持近单目标效率，同时忠实维护个体身份。

## 2. 方法论

### 核心思想
- 基于SAM2，将模型改造为**交互式多目标视频分割框架**。使用**显式目标查询**并行表示不同个体，同时采用**共享全局表示**建模上下文。通过**解耦掩码注意力**防止跨目标干扰，利用**基于查询的稀疏记忆**实现稳定时序演化。最终实现延迟与目标数量解耦。

### 关键技术细节
1. **可扩展目标查询**：为用户指定的每个目标分配一个ID感知的查询，自然适应不同目标数量。
2. **解耦掩码注意力**（Decoupled Masked Attention）：
   - 在自注意力层中构造注意力掩码，使得：
     - 全局查询与所有目标查询可见；
     - 同一目标的多个查询之间可见；
     - 不同目标的查询之间不可见（设为−∞）。
   - 公式：`Q′ = softmax(QK^T + M)V + Q`，其中M的条目根据目标索引范围定义。
3. **查询合并**（初始帧）：对每个目标的多点（Nq个点）通过MLP加权头计算重要性权重，合并为单个目标查询。后续帧每个目标仅用一个查询。
4. **基于查询的稀疏记忆**：
   - 仅用占全局掩码的密集记忆捕捉全局上下文；为每个目标保存**轻量查询**（每帧每目标仅1 token）到FIFO稀疏记忆，时间窗口大小为T帧（默认16），大幅降低存储成本。
   - 公式：`M_sparse,t = [ˆQ_t, ˆQ_{t-1}, ..., ˆQ_{t-T+2}, ˆQ_0] ∈ R^{T·k×C}`。
5. **身份变换器**（Identity Transformer）：
   - 利用稀疏记忆更新目标查询。上一帧目标查询`ˆQ_{t-1}`交叉注意力到稀疏记忆，并应用身份掩码确保每个查询只关注自身历史，避免身份漂移。
   - 公式：`Q_t = softmax(ˆQ_{t-1} M_sparse,t-1^T + ˜M) M_sparse,t-1 + ˆQ_{t-1}`，其中˜M限制每个目标只关注自身索引。
6. **训练策略**：
   - 两阶段训练：阶段I静态图像训练（高质量单次多目标分割），阶段II视频训练（时序身份一致性）。
   - **步进采样**（Strided Sampling）：混合连续帧和4帧间隔的步进采样（跨度32帧），处理长时序遮挡与重现。
   - **交叠损失**（Overlap Prevention）：惩罚不同目标预测掩码的重叠区域，减少像素级歧义。
   - 损失函数：个体掩码损失（Focal + Dice + 交叠损失）+ 全局掩码损失（Focal + Dice + IoU + 目标得分损失）。

## 6. 主要结论与发现

- SAM-MT在**6个VOS benchmark**上达到或超越SOTA（如MOSEv2 J&˙F 43.0，比SAM2.1-B+高1.9；LVOSv2 J&F 76.6，高2.0），证明零样本泛化能力。

- **步进采样和交叠损失**有效提升长时序遮挡处理和密集场景身份区分。
- **解耦掩码注意力**是保持身份独立的关键：全可见或全遮盖均导致性能大幅下降（J&˙F降5.5和3.7）。

## 7. 优点

- **框架级创新**：首次在SAM族方法中系统解决多目标瓶颈，从架构上解耦计算与目标数。
- **高效设计**：轻量目标查询（每帧每目标仅1 token）代替密集像素记忆，稀疏记忆实现长时序（窗口16帧）而计算开销极低。
- **身份保持机制**：解耦掩码注意力+身份掩码的稀疏记忆更新，有效避免跨目标干扰和身份漂移。
- **训练策略实用**：步进采样模拟长时遮挡，交叠损失强制空间分离，均为实际挑战而设计。
