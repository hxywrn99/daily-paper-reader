## 摘要
当使用所有可用序列时，多对比3D MRI分割的计算需求可能很高。我们评估了一种预训练的部分信息分解框架，该框架根据输入对关于区域肿瘤负荷的冗余、独特和协同信息进行排序，并为下游训练选择排名最高的对。应用于T1n、T1c、T2w和T2-FLAIR MRI，该框架选择了T1c+T2-FLAIR。然后，我们使用不同的输入配置训练了十一个架构相同的轻量级3D U-Net。在一个独立的测试队列上，T1c+T2-FLAIR是最强的两输入配置，在平均Dice系数上总体排名第二（0.676对比所有四个输入的0.687）。对全输入模型的独立Shapley分析也确认了T2-FLAIR和T1c是最有影响力的输入，且它们的成对交互作用最强。这些发现证明了基于PID的预训练选择在开发昂贵的3D模型之前识别紧凑、信息丰富的MRI输入集方面的实用价值。

## Abstract
Multi-contrast 3D MRI segmentation can be computationally demanding when all available sequences are used. We evaluate a pre-training Partial Information Decomposition framework that ranks input pairs according to their redundant, unique, and synergistic information about regional tumor burden and selects the highest-ranked pair for downstream training. Applied to T1n, T1c, T2w, and T2-FLAIR MRI, the framework selected T1c+T2-FLAIR. We then trained eleven architecturally identical lightweight 3D U-Nets using different input configurations. On an independent test cohort, T1c+T2-FLAIR was the strongest two-input configuration and ranked second overall in mean Dice (0.676 versus 0.687 for all four inputs). Independent Shapley analysis on the full-input model also identified T2-FLAIR and T1c as the most influential inputs and their pairwise interaction as the strongest. These findings demonstrate the practical value of PID based pre-training selection for identifying compact, informative MRI input sets before costly 3D model development.

- **整体含义**：提出一种基于部分信息分解（PID）的预训练选择策略，作为资源受限场景下多对比 MRI 输入降维的实用解决方案。

### 2. 论文提出的方法论
- **核心思想**：利用 PID 将输入对关于区域肿瘤负荷（水肿、增强肿瘤、坏死）的总信息分解为冗余、独特和协同三部分，并设计聚合得分来量化每对输入组合的信息价值，选择得分最高的一对用于下游训练。
- **关键技术细节**：
    1. **数据预处理**：从训练集中每个受试者采样 128 个空间对齐的 32×32×32 体素块（70% 来自肿瘤前景）。对每个肿瘤区域 c，将体素块中的肿瘤负荷（阳性体素占比）离散化为 4 类（零负担 + 三个分位数类）。
    2. **自动编码器与量化**：为每个 MRI 序列独立训练一个浅层 3D 自动编码器（编码器：3 个 stride=2 的 3×3×3 卷积块，输出通道 8/16/32，经实例归一化和 ReLU，展平后投影到 32 维潜变量）。训练后冻结编码器，对每个序列的潜变量用 K-means（K=16）聚类，将连续潜变量离散化为簇标签（源编码 q）。
    3. **PID 计算**：对每个输入对 (a,b) 和每个肿瘤区域 c，基于经验联合分布计算互信息，并采用最小互信息（MMI）PID 公式分解：
       - \( R^c_{ab} = \min\{I(q_a;T_c), I(q_b;T_c)\} \)
       - \( U^c_{a|b} = I(q_a;T_c) - R^c_{ab} \)，\( U^c_{b|a} = I(q_b;T_c) - R^c_{ab} \)
       - \( S^c_{ab} = I((q_a,q_b);T_c) - R^c_{ab} - U^c_{a|b} - U^c_{b|a} \)
    4. **聚合得分**：\( \text{Score}(a,b) = \sum_c w_c [ I((q_a,q_b);T_c) + \lambda (U^c_{a|b} + U^c_{b|a} + S^c_{ab}) ] \)，其中 \( \lambda = 0.5 \)，\( w_c \) 为区域 c 阳性块频率的逆归一化权重。
    5. **选对策略**：选择得分最高的二元组 \( M^* = \arg\max \text{Score}(a,b) \)。
- **下游模型**：轻量级 3D U-Net（三层分辨率，每个编码/解码块含两个 3×3×3 卷积 + ReLU + 实例归一化；最大池化下采样，最近邻插值 + 1×1×1 卷积上采样）。输入 64×64×64 体素块，batch size=1，每 epoch 256 块，训练 200 轮，使用 Dice+BCE 损失，AdamW 优化器（weight decay=1e-4），余弦学习率调度（1e-3 → 1e-5）。

### 3. 实验设计
- **数据集**：CoRe-BT 数据集中的 MRI 和肿瘤掩膜部分。筛选后使用 132 训练、30 验证、28 测试受试者。输入为 T1n、T1c、T2w、T2-FLAIR 四个序列。目标区域：水肿、增强肿瘤、坏死/非增强肿瘤。
- **基准与对比**：
  - 训练 11 个架构完全相同的 3D U-Net：1 个四输入模型、全部 6 个两输入组合、全部 4 个单输入模型。
  - 评价指标：受试者层面宏平均 Dice、HD95、敏感度、精确度。
  - 统计方法：10,000 次 bootstrap 重采样、Spearman 相关、配对 Wilcoxon 检验、Friedman 检验 + Holm 校正、-0.03 Dice 非劣效性界限。

1. **PID 选择有效**：PID 得分排名第一的 T1c+T2-FLAIR 在测试中实际是最强的两输入配置（平均 Dice 0.676），仅次于四输入模型（0.687），保留了 98.5% 的 Dice 性能且减半输入通道。

3. **临床可解释性**：T1c 直接突出增强肿瘤，T2-FLAIR 敏感于水肿和浸润性异常，两者互补覆盖了主要目标区域。
4. **资源节省**：预训练选择仅需训练 n 个轻量自动编码器并评估 \(\binom{n}{2}\) 个得分，避免了穷举下游模型训练，显著降低计算和存储开销。

### 7. 优点
- **创新性**：首次将 PID 的冗余/独特/协同分解系统性地应用于多对比 MRI 序列的预训练选择，并提出区域感知聚合得分。
- **计算效率**：选择过程仅需预训练自动编码器和 K-means 量化，远低于全序列训练成本；且选择结果可复用至不同下游架构。
- **验证完整性**：同时采用分数排名关联、独立统计检验、非劣效性比较、事后 Shapley 归因等多角度验证，增强结论可靠性。
- **可解释性**：PID 分解成分（冗余、独特、协同）直接对应临床直觉（哪些序列提供独特信息、哪些联合效果更强）。

### 8. 不足与局限
- **实验规模较小**：仅有 4 个输入序列、6 个两输入对，统计检验功效不足，无法得出显著差异结论。

- **架构通用性未验证**：仅测试轻量 3D U-Net，未在高性能模型（如 nnU-Net、ViT）上验证选择策略的有效性。
