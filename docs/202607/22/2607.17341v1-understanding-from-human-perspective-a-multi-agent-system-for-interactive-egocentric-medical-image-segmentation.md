## 摘要
交互式自我中心医学图像分割（IEMIS）在智能眼镜辅助的医学图像审阅中扮演重要角色，能从临床医生的自我中心视角分割其提及的医学目标。一旦成功，其提供的目标级视觉证据能强化审阅过程，并为细粒度分析和临床决策提供支持。然而，指令和视频均来自用户的自我中心视角，这带来了两个挑战：（1）语义歧义导致模型无法确认用户意图的目标；（2）视觉变异性使得分割在帧间跳跃。本文提出EgoMed-Agent，一个多智能体系统，通过两个工作流程从人类视角理解目标：（1）目标确认工作流将指令与候选目标进行接地，并附带可靠性评分，当接地可靠时确认目标，否则请求用户澄清，从而确认分割目标；（2）定位引导传播工作流将掩码传播与逐帧目标定位耦合，当两者出现偏差时利用定位目标纠正传播的掩码，从而在整个自我中心视频中保持分割目标的一致性。大量实验表明，EgoMed-Agent的平均Dice达到71.34%，远超最佳文本提示基线（11.70%）。我们的代码可在项目页面获取。

## Abstract
Interactive egocentric medical image segmentation (IEMIS) plays an important role in smart-glasses-assisted medical image review, segmenting the medical targets a clinician refers to from their egocentric view. Once it succeeds, the object-level visual evidence it provides strengthens the review and underpins fine-grained analysis and clinical decision-making. However, the instruction and the video both come from the user's egocentric perspective, which poses two challenges. (1) Semantic ambiguity leaves the model unable to confirm the user-intended target. (2) Visual variability makes the segmentation jump from frame to frame. In this paper, we propose EgoMed-Agent, a multi-agent system that understands the target from the human perspective through two workflows. (1) The \textit{Target Confirmation Workflow} grounds the instruction against candidate targets with a reliability score, confirming the target when the grounding is reliable and asking the user to clarify when it is not, thereby confirming the segmentation target. (2) The \textit{Localization-Guided Propagation Workflow} couples mask propagation with per-frame target localization, using the localized target to correct the propagated mask whenever the two diverge, so the segmentation stays on the target across the egocentric video. Extensive experiments show that EgoMed-Agent reaches 71.34\% average Dice, far above the best text-prompted baseline (11.70\%). Our code is available at \href{https://github.com/wdyyyyyy/EgoMed-Agent}{our project page}.

- **核心问题**：交互式自我中心医学图像分割（IEMIS）旨在从医生的自我中心视角（通过智能眼镜拍摄）分割用户指令中提到的医学目标。然而，由于指令和视频均来自用户的自我中心视角，面临两大挑战：

  - **视觉变异性**：头部运动、视角变化和光照变化导致目标在帧间位置、尺度、形状不断变化，使得分割结果在帧间跳跃。
- **研究动机**：现有的文本提示分割方法直接映射指令到掩码，无法处理歧义和漂移，在自我中心医学场景下性能极低（平均Dice不足12%）。因此需要一种能够理解用户意图并保持分割稳定性的新范式。
- **整体含义**：本文提出多智能体系统EgoMed-Agent，通过分解任务为“目标确认”和“定位引导传播”两个工作流，从人类视角理解目标，从而为智能眼镜辅助的医学图像回顾提供可靠、稳定的交互式分割，支撑细粒度分析和临床决策。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：采用多智能体系统（MAS）将IEMIS分解为两个耦合的工作流，由三个专门智能体协作完成：
  - **目标确认工作流**（Detection Agent + Confirmation Agent）：先提取候选目标，再将指令与候选目标接地并评估可靠性，确认意图目标或请求用户澄清。
  - **定位引导传播工作流**（Detection Agent + Propagation Agent）：利用检测器逐帧定位目标，并与掩码传播结果进行一致性评估，在漂移时用定位框重初始化传播，保持分割对齐。
- **关键技术细节**：
  - **Detection Agent**：包含图像类型分类器（YOLO26-classify）和模态专用检测工具箱（YOLO26-det，分别对应CT、MRI、超声、X光、内窥镜）。先识别当前帧的成像类型，选择相应检测器，输出候选目标（类别+边界框）。
  - **Confirmation Agent**：基于大语言模型（DeepSeek-V4-Flash），将指令与候选目标集合进行语义接地，输出目标结果θ*和可靠性分数s。若s≥τ1（实验中τ1=0.8），则直接确认目标；若s<τ1，则主动请求用户澄清，根据反馈获取明确目标θr。
  - **Propagation Agent**：基于SAM2（冻结权重），从Detection Agent提供的初始定位框出发，生成初始掩码并向前传播，得到连续掩码序列。
  - **一致性评估**：在每帧，计算检测框与传播框的IoU。若IoU≥τ2（实验中τ2=0.6），则保持传播；若IoU<τ2，则用检测框重初始化Propagation Agent，纠正漂移。
- **整体流程公式**（文字说明）：
  - 目标确认：θ = W1(v1, I; Adet, Acon)
  - 掩码序列生成：M1:t = W2(V1:t, θ; Adet, Apro)

## 6. 论文的主要结论与发现

- EgoMed-Agent在12个目标上平均Dice达71.34%，远超最佳文本提示基线MedSAM3（11.70%），p<0.001。
- 直接使用文本指令进行开放词汇分割在自我中心医学场景下极为不可靠（基线大多<12%），因为缺乏目标确认步骤。
- 目标确认工作流有效解决语义歧义：可靠接地时直接确认，不可靠时通过澄清使意图明确（使用强LLM达到100%准确）。
- 定位引导传播工作流通过耦合传播与检测纠正漂移，比单纯传播或单纯检测分别提升约21%和12%平均Dice。
- 一致性阈值τ2=0.6取得最佳平衡；LLM的选择重要，DeepSeek-V4-Flash显著优于DeepSeek-Chat（尤其处理空间关系）。
- 系统在单一GPU上可实现近实时（10.7 FPS），具备实际部署潜力。
