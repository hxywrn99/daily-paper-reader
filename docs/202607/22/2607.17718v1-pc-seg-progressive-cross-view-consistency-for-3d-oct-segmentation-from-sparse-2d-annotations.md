## 摘要
光学相干断层扫描（OCT）图像的体积分割对于诊断眼部疾病至关重要，但需要耗费大量人力的逐体素标注。虽然半监督学习（SSL）可以降低标注成本，但大多数现有方法逐切片处理数据，未能利用固有的三维空间上下文。我们提出PC-Seg，一种渐进式跨视图一致性框架，能够从稀疏的二维标注中学习高精度的三维分割模型。与传统的多视图方法不同，PC-Seg使用单一的二维模型，从标准B扫描和正交切片中学习跨视图一致性，从而生成可靠的三维伪标签。然后将这些伪标签蒸馏到三维模型中，随后进入协同训练阶段，二维和三维模型通过集成伪标签相互细化。在MSHC和Duke DME数据集上的实验表明，PC-Seg仅使用约0.7%的训练数据标注就能达到与全监督学习相当的精度，优于最先进的半监督和视网膜层分割方法。我们的代码已公开于https://github.com/gsisaoki/pc-seg-official。

## Abstract
Volumetric segmentation of optical coherence tomography (OCT) images is essential for diagnosing ocular diseases but requires labor-intensive voxel-wise annotations. While semi-supervised learning (SSL) can reduce annotation costs, most existing methods process data slice by slice and fail to exploit the inherent 3D spatial context. We propose PC-Seg, a progressive cross-view consistency framework that learns high-accuracy 3D segmentation models from sparse 2D annotations. Unlike conventional multi-view approaches, PC-Seg uses a single 2D model to learn cross-view consistency from standard B-scans and orthogonal slices, thereby generating reliable volumetric pseudo-labels. These pseudo-labels are then distilled into a 3D model, followed by a co-training stage in which the 2D and 3D models mutually refine each other through ensemble pseudo-labeling. Experiments on the MSHC and Duke DME datasets demonstrate that PC-Seg achieves accuracy comparable to fully supervised learning while using labels for only about 0.7% of the training data, outperforming state-of-the-art semi-supervised and retinal layer segmentation methods. Our code is publicly available at https://github.com/gsisaoki/pc-seg-official.

## 1. 论文的核心问题与整体含义（研究动机和背景）

- **研究动机**：OCT（光学相干断层扫描）体积分割是诊断青光眼、糖尿病视网膜病变等眼科疾病的关键生物标志物，但获取逐体素（voxel-wise）的人工标注成本极高。现有的半监督学习方法大多按2D切片逐张处理，忽略了OCT数据固有的三维空间上下文信息，分割精度受限。
- **核心挑战**：OCT体积具有强各向异性（B-scan内分辨率高，层间间距大），且标注通常仅限于标准的2D B-scan切片，正交方向（orthogonal）缺少标注。直接从少量2D标注学习3D分割模型容易过拟合、误差累积。
- **论文目标**：提出一种新型框架PC-Seg，仅利用稀疏的2D标注（约0.7%训练数据）即可训练出与全监督方法精度相当的高精度3D分割模型，大幅降低标注成本。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：采用**渐进式课程学习（curriculum learning）**策略，从单个2D模型出发，依次学习标准B-scan和正交B-scan的一致性，生成可靠的体积伪标签，再蒸馏到3D模型，最后通过2D/3D协同训练相互精化。
- **五个阶段流程**：
  1. **Stage 1 – 2D预热**：使用标注和未标注的B-scan图像，基于UniMatch（弱‑到‑强一致性正则化）训练一个2D ResUNet，生成所有B-scan的初始伪标签。
  2. **Stage 2 – 适应正交切片**：引入未标注的正交B-scan，利用Stage 1的伪标签计算一致性损失（包括弱、特征级、两种强扰动共四种），训练2D模型适应正交视图，并生成正交B-scan的伪标签。
  3. **Stage 3 – 正交交叉教学（Cross‑teaching）**：在B-scan和正交B-scan之间互相生成伪标签（即用B-scan预测正交视图像素，反之亦然），通过交叉教学提升跨视图一致性，最终将两个平面的预测集成生成高质量的3D伪标签。
  4. **Stage 4 – 知识蒸馏到3D模型**：基于Stage 3生成的3D伪标签，对3D ResUNet进行半监督训练（伪标签中已知标注的切片替换为真实标签）。
  5. **Stage 5 – 2D/3D协同训练**：交替训练2D和3D模型，并将两者的预测集成更新3D伪标签，实现相互精化。
- **关键技术细节**：
  - 采用ResUNet（2D和3D版本）作为基线架构，深度4，基础通道数32。CNN在实验中优于Transformer（如SegFormer、Swin-UNet、UNETR），且参数更少。
  - 使用UniMatch框架，其中弱扰动（Aw）包括随机裁剪、水平翻转、垂直缩放及OCT专用增强（FDDA和PRLC）；强扰动包括颜色抖动、模糊、缩放等；特征级扰动为dropout（比例0.5）。
  - 伪标签置信度阈值τ初始设为0.95，但在Stage 4/5对病变类别动态降至0.5，因为集成预测更鲁棒。
  - 除Stage 1外，后续阶段均根据验证集最优权重动态更新伪标签，减少误差累积。
