## Abstract
With the continuous improvement of device imaging resolution, the popularity of Ultra-High-Definition (UHD) images is increasing. Unfortunately, existing methods for fusing multi-exposure images in dynamic scenes are designed for low-resolution images, which makes them inefficient for generating high-quality UHD images on a resource-constrained device. To alleviate the limitations of extremely long-sequence inputs, inspired by the Large Language Model (LLM) for processing infinitely long texts, we propose a novel learning paradigm to achieve UHD multi-exposure dynamic scene image fusion on a single consumer-grade GPU, named Infinite Pixel Learning (IPL). The design of our approach comes from three key components: The first step is to slice the input sequences to relieve the pressure generated by the model processing the data stream; Second, we develop an attention cache technique, which is similar to the KV cache for infinite data stream processing; Finally, we design a method for attention cache compression to alleviate the storage burden of the cache on the device. In addition, we provide a new UHD benchmark to evaluate the effectiveness of our method. Extensive experimental results show that our method maintains high-quality visual performance while fusing UHD dynamic multi-exposure images in real-time (>40fps) on a single consumer-grade GPU.

## 1. 论文的核心问题与整体含义

- **研究动机**：随着设备成像分辨率持续提升，超高清（UHD）图像越来越普及。然而，现有的动态场景多曝光图像融合方法主要针对低分辨率图像设计，在资源受限设备（如消费级GPU）上处理多个UHD图像时，会出现显存溢出、推理速度慢、融合质量低等问题，难以实现高质量UHD实时融合。
- **核心问题**：如何在大分辨率、多输入（通常为3张不同曝光图像）的场景下，在单块消费级GPU上实现高精度、实时（>40fps）的动态多曝光图像融合，并有效避免鬼影（ghosting）和模糊。
- **整体含义**：受大语言模型（LLM）处理无限长文本的“分块-缓存-量化”流水线启发，论文提出**无限像素学习（Infinite Pixel Learning, IPL）**，将UHD图像视为像素序列进行分块处理，并引入注意力缓存与量化压缩技术，首次实现了在消费级GPU上对全分辨率UHD动态多曝光图像的高效融合，兼顾速度与质量。

## 2. 论文提出的方法论

- **核心思想**：模仿LLM中处理长序列的技术（分块、KV缓存、量化），将多张UHD图像沿通道、宽度、高度三个维度切分为小块，通过循环扫描器逐块提取局部特征，并利用**注意力缓存**存储中间特征以避免重复计算，再用**量化压缩**减轻缓存存储负担；同时设计**维度滚动变换模块**来补偿因分块导致的全局特征损失。
- **关键技术细节**：
  1. **分块循环扫描器（Slice Cyclic Scanner）**  
     - 沿通道、宽度、高度三个维度将输入特征图切分成块。
     - 每个维度上使用循环扫描器逐块送入局部特征提取器（LFE）。
     - LFE结构：全局平均池化 + 1×1卷积 → ReLU → 1×1卷积 → Sigmoid 生成通道注意力权重，与原始特征逐元素相乘。
  2. **注意力缓存（Attention Cache）**  
     - 推理时，首次计算得到的LFE输出存入缓存，后续相同块直接读取缓存结果，无需重复卷积计算，加速推理。
     - 公式：若块k在缓存中，则Fk = CR(k)；否则Fk = LFE(k)，并执行写操作CW(k)。
  3. **量化压缩（Quantization Compression）**  
     - 对缓存中的浮点张量进行对称量化（计算缩放因子s和零点zp），将32位浮点映射为低精度整数（如int8），减少存储占用。
     - 读取时通过解码器恢复为浮点数用于计算。
  4. **维度滚动变换模块（DRTM）**  
     - 对DAEM输出的特征图进行插值调整分辨率，然后通过三次“1×1卷积 + GELU + 维度置换（Permute）”操作，从通道、宽度、高度三个视角变换特征图，建立跨维度长距离依赖。
     - 最后通过插值恢复原始分辨率，与输入逐元素相乘得到全局增强特征。
- **整体流程**：
  - 输入三张不同曝光图像（X1, X2, X3），拼接后经下采样器（DownSampler）降维到48通道特征空间。
  - 经过8个级联的特征集成块（FIB），每个FIB包含DAEM和DRTM，两者输出相加。
  - 最终特征经上采样器（UpSampler）恢复原始分辨率，与参考图像X2逐元素乘积得到融合图像Ŷ。
  - 损失函数：L1损失（RGB空间）。

## 6. 论文的主要结论与发现

- **主要结论**：提出的IPL方法能够在单块消费级GPU上，实现全分辨率（3840×2160）UHD动态多曝光图像实时融合（>40fps），并显著优于现有SOTA方法（PSNR提升约46%，SSIM提升约48%，LPIPS降低约88%）。在非UHD数据集上，IPL也达到与SOTA可比的性能，验证了其通用性。
- **关键发现**：
