## 摘要
特征融合网络是现代目标检测器的基础组件，通过聚合多尺度特征来检测不同大小的物体。然而，由于不同金字塔层的特征表示异构，直接融合它们往往会导致语义不一致。本文提出特征交互网络（FINE），这是一种轻量级语义对齐模块，通过跨层级注意力在融合前利用高层上下文引导来细化低层特征。为了弥合结构差距并确保计算效率，我们引入了对齐感知令牌采样，该采样对齐不同尺度间的对应空间区域，将注意力复杂度降低一个数量级。由此产生的注意力权重生成一个空间-通道调制图，该图通过上采样并经由残差逐元素调制应用于低层特征。这种机制确保网络选择性地增强语义相关像素，同时保留密集预测任务所需的亚像素定位精度。FINE可普遍应用于各种检测器，并在不牺牲效率的情况下持续提升检测精度。

## Abstract
Feature fusion networks are fundamental components in modern object detectors, aggregating multi-scale features to detect objects of varying sizes. However, directly fusing features from different pyramid levels often introduces semantic inconsistency due to their heterogeneous representations. In this paper, we propose Feature Interaction NEtwork (FINE), a lightweight semantic alignment module that refines low-level features via high-level contextual guidance using cross-level attention prior to fusion. To bridge the structural gap and ensure computational efficiency, we introduce an Alignment-Aware Token Sampling that aligns corresponding spatial regions across scales, reducing the attention complexity by an order of magnitude. The resulting attention weights generate a spatial-channel modulation map that is upsampled and applied to the low-level features via residual element-wise modulation. This mechanism ensures that the network selectively enhances semantically relevant pixels while preserving the sub-pixel localization accuracy necessary for dense prediction tasks. FINE is generally applicable to various detectors and consistently improves detection accuracy without compromising efficiency.

## 1. 核心问题与整体含义（研究动机和背景）
- **核心问题**：特征金字塔网络（FPN）等特征融合网络在聚合多尺度特征时，由于不同金字塔层的特征表示存在**语义不一致**（semantic inconsistency）。高层特征语义丰富但空间粗糙，低层特征细节丰富但语义不足。直接通过简单的逐元素操作（如加法或拼接）融合会导致信息冲突，降低目标检测精度，尤其在背景杂乱和小物体检测时容易出现误分类和漏检。
- **研究动机**：设计一种轻量级、可即插即用的模块，在融合前利用高层语义上下文引导低层特征，实现跨层语义对齐，从而提升检测性能而不影响效率。
- **整体含义**：提出**特征交互网络（Feature Interaction NEtwork, FINE）**，通过在特征金字塔的不同层级间引入对齐感知的跨层注意力，有效缓解语义错位，显著提升多种检测器（实时检测器和经典检测器）的准确率，且计算开销极小。

## 2. 方法论：核心思想、关键技术细节
- **核心思想**：在融合操作之前，利用高层特征（`F_high`）的丰富语义通过**跨层注意力**生成空间-通道调制图，对低层特征（`F_low`）进行细化，得到语义对齐的低层表示（`F_aligned_low`），再与上采样后的高层特征融合。
- **关键技术细节**：
  - **对齐感知令牌采样（Alignment-Aware Token Sampling, AATS）**：
    - 为了解决高层与低层特征之间有效感受野（ERF）不匹配以及计算复杂度过高的问题，AATS 使用非对称采样核（`kernel = k` 用于高层，`kernel = rk` 用于低层，其中 `r` 为对齐采样比，通常设为2）对两层特征进行下采样，使它们的 ERF 对齐，同时大幅减少令牌数量。
    - 默认使用无参数的平均池化，计算开销低。
  - **瓶颈多头跨层注意力（Bottleneck Multi-Head Cross-Level Attention）**：
    - 将下采样后的低层特征作为 Query，下采样后的高层特征作为 Key 和 Value，通过标准的多头自注意力（Multi-Head Self-Attention）机制计算跨层注意力。
    - 注意力输出经 reshape 和上采样后得到空间-通道调制图 `M`，分辨率与原始 `F_low` 一致。
  - **残差空间-通道调制（Residual Spatial-Channel Modulation）**：
    - 将调制图 `M` 与原始 `F_low` 进行逐元素相乘（`F_low ⊙ M`），再加上原始 `F_low` 作为残差连接，得到对齐后的特征：`F_aligned_low = (F_low ⊙ M) + F_low`。
    - 该设计既能注入高层语义上下文，又保留低层的精细空间细节，避免粗粒度的调制图破坏定位精度。
- **算法流程（文字描述）**：
  - 输入：相邻两层特征 `F_low` (stride = `s`) 和 `F_high` (stride = `2s`)。
  - 步骤1：通过 AATS 对 `F_low` 和 `F_high` 进行下采样，得到 `F_hat_low` 和 `F_hat_high`。
  - 步骤2：将 `F_hat_low` 展平作为 Q，`F_hat_high` 展平作为 K、V，计算多头交叉注意力，输出调制图 `M_hat`。
  - 步骤3：将 `M_hat` 上采样至 `F_low` 分辨率得到 `M`。
  - 步骤4：执行残差调制：`F_aligned_low = (F_low ⊙ M) + F_low`。
  - 步骤5：将 `F_aligned_low` 与上采样后的 `F_high` 进行融合（加法或拼接）。

## 6. 论文的主要结论与发现
- **FINE 模块能够显著降低特征融合中的语义不一致**，在几乎所有检测器上提升 AP，尤其对小物体（APs）提升最为显著（相对提升 4.8%~10.4%）。
- **效率优势**：AATS 将注意力复杂度从二次方降低近一个数量级，使 FINE 的额外 FLOPs 仅增加 1.0~1.2G，参数量增加约 1M，推理速度几乎不受影响。
- **假阳性大幅减少**：在 Mask R-CNN 上 FP 减少 9.93%，召回率提升 1.52%，F1-score 提升 3.77%。
- **通用性**：FINE 不仅适用于目标检测，还能提升实例、语义、全景分割的性能。
- **残差调制优于直接调制**：保留低层空间细节，对小物体定位更友好。

## 7. 优点
- **轻量即插即用**：模块设计简洁，可无缝嵌入现有任何 FPN/PAN 类检测器，不改动训练流程。
- **高效跨层注意力**：AATS 通过感受野对齐大幅降低计算量，使交叉注意力在实时检测器中可行。
- **残差调制保留细节**：避免粗粒度调制图破坏精细空间信息，特别有利于小物体检测。
